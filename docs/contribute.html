Ways to Contribute

FlatFree is a community project and we welcome contributions of all kinds. Here's how you can help:

Submit Apps

Package your favorite free software as Flatpaks and submit them to FlatFree. Our CI handles the rest.

Improve Docs

Help us write better documentation, guides, and FAQs for the FlatFree community.

Spread the Word

Tell other Linux users about FlatFree. The more people know, the bigger our community grows.

Report Issues

Found a bug or have a suggestion? Open an issue on our GitHub repository.

Contribute on GitHub
