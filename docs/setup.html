Setting Up FlatFree

Follow these steps to start using FlatFree on your Linux system.

1. Ensure Flatpak is Installed

Most Linux distributions include Flatpak. Install it if needed:

# Debian/Ubuntu
sudo apt install flatpak

# Fedora
sudo dnf install flatpak

# Arch Linux
sudo pacman -S flatpak

# openSUSE
sudo zypper install flatpak

2. Add the FlatFree Remote

flatpak remote-add --if-not-exists flatfree https://natalie.acreetionos.org/FlatFree/flatfree.flatpakrepo

3. Install an Application

flatpak install flatfree org.acreetionos.MediaWriter

4. Run the Application

flatpak run org.acreetionos.MediaWriter

Or launch it from your desktop's application menu.

5. Keep Apps Updated

flatpak update
