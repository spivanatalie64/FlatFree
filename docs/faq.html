Frequently Asked Questions

Everything you need to know about FlatFree

What is FlatFree?

FlatFree is a community-maintained Flatpak repository that accepts all free and open source software, regardless of how it was created. We are the alternative to Flathub's exclusionary policies regarding AI-assisted development.

How is FlatFree different from Flathub?

Flathub bans applications "generated predominantly by AI." FlatFree welcomes them. We believe that any tool used by a human is human work, and we do not police how code is written. If it builds, if it's free software — it belongs here.

How do I install apps from FlatFree?

First add the remote: flatpak remote-add --if-not-exists flatfree https://natalie.acreetionos.org/FlatFree/flatfree.flatpakrepo. Then install: flatpak install flatfree <app-id>.

How do I submit an application?

Open a pull request on our GitHub repository adding a directory with your app ID and a flatpak-builder manifest. Our CI validates the manifest, builds the app, and runs a malware scan. If it passes, it's in — no human reviewers needed.

Can I submit apps created with AI assistance?

Absolutely. FlatFree fully accepts and welcomes submissions that use AI assistance. We do not believe in rejecting software because of who wrote it or how it was made. All free software is welcome here.

Is FlatFree affiliated with Flathub or the Flatpak project?

No. FlatFree is an independent community project. We are not affiliated with Flathub, the Flatpak project, or any corporation. We are simply people who believe in free software without gatekeeping.

What license do my apps need to be?

All applications must be free and open source software. We accept any OSI-approved open source license. Proprietary software is not accepted.

Who maintains FlatFree?

FlatFree is maintained by Natalie Cole-Clift Spiva with contributions from the community. We are independent developers who believe in free software for everyone.

FlatFree is not affiliated with Flathub or the Flatpak project.
