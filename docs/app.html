Description

AcreetionOS Media Writer is a straightforward utility for writing operating system images to USB drives. Forked from Fedora Media Writer and rebranded for AcreetionOS, it provides a clean interface for creating bootable installation media.

Built with Qt 6 and designed for Linux, it supports writing ISO and raw image files to USB flash drives. No complicated command-line options — just select your image, select your drive, and write.

This application is free software, licensed under the GPL, and welcomes contributions from all developers regardless of how they build their software.

Install

Add the FlatFree remote if you haven't already:

flatpak remote-add --if-not-exists flatfree https://natalie.acreetionos.org/FlatFree/flatfree.flatpakrepo

Then install this application:

flatpak install flatfree org.acreetionos.MediaWriter

Or run it directly once installed:

flatpak run org.acreetionos.MediaWriter
