Welcome to FlatFree

Back to Blog · June 13, 2026 by Natalie

Today, I'm thrilled to announce the public launch of FlatFree — a new community Flatpak repository built on a radically simple principle: if it's free software and it builds, it belongs here.

FlatFree is not just another package repository. It's a statement. It's a response to a growing problem in the open source ecosystem: gatekeeping disguised as quality control.

Our Mission

FlatFree exists to be an open Flatpak repository that welcomes all free software, regardless of how it was created. Whether you write every line by hand, use AI assistants, or generate code through entirely automated means — if the result is free software licensed under an OSI-approved license, you have a home here.

We believe that the four essential freedoms of free software — the freedom to use, study, modify, and share — are about the software itself, not the method of its creation. Judging software by how it was made rather than what it does is antithetical to the spirit of open source.

How We're Different

Unlike other repositories that rely on subjective human review panels, FlatFree uses automated checks:

  • Manifest validation — our CI confirms your flatpak-builder manifest is well-formed
  • Automated builds — we actually build your application to verify it compiles
  • Malware scanning — we run security checks to protect users
  • License verification — we confirm the software uses an approved free license

Pass these objective checks and you're in. No reviewers, no subjective judgments, no moral policing.

Getting Started

Adding FlatFree as a remote is straightforward. Open a terminal and run:

flatpak remote-add --user flatfree https://flatfree.acreetionos.org/flatfree.flatpakrepo

Once added, you can install applications just like you would from any other remote:

flatpak install --user flatfree com.example.MyApp

Browse the apps page to see what's available, or check the setup guide for detailed instructions.

What's Next

FlatFree is just getting started. Right now we have a small but growing collection of applications, a fully functional Flatpak remote, and an automated CI pipeline ready to accept new submissions. The infrastructure is in place — what we need now is the community.

If you have a free software application — especially one that Flathub has rejected or that you're worried they might reject — we want it. Open a pull request on our GitHub repository and submit your manifest.

If you believe in our mission, spread the word. Star the repository, tell other developers, contribute to the website, help us build the repository that open source deserves.

Welcome to FlatFree. The alternative has arrived.

FlatFree is not affiliated with Flathub or the Flatpak project.
